## prerequisite

- Docker [docker](https://docs.docker.com/get-docker/)

## Running the app

```bash
# Clone repo
$ git clone https://github.com/yengobilly4063/ultragame
$ cd ultragame

# Start
$ docker-compose up

$Stop and Remove running containers
$ ctrl && c|z
$ docker-compose down

$ api url : http://localhost:300
```
